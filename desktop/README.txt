Jeff订单工具使用说明

1. 双击“打开Jeff订单工具.exe”打开工具。
2. 工具会自动启动后台服务，并打开浏览器页面。
3. 第一次打开会提示设置管理员密码；以后电脑和手机访问都先输入这个密码。
4. 登记订单时先输入订单号，再选择公司、工厂并填写各类数量；如已先交一部分，可同时填写“首批先交”。
5. 以后再次先交时，在订单详情的“记录本次先交”中填写数量；工具会自动显示累计先交和剩余未交。
6. 首页“待核销明细”会显示当前剩余未交的套装、单衫、单裤、马甲和大衣数量。
7. 订单总表默认按订单号从小到大排列，也可改为按登记日期排序。
8. 电脑端订单详情使用紧凑横排；“记录本次先交”默认收起，需要时点击展开填写。
9. 数量汇总会按“套装 128”“单衫 98”这种方式显示，品类和数字是一组。
10. 用完后可以直接关闭浏览器；需要关闭后台服务时，打开 SupportFiles 文件夹，双击 CloseJeffOrderTool.exe。
11. 数据保存在当前文件夹的 data 目录。
12. 备份时复制整个 data 文件夹，不要只复制单个 orders.db。
13. 如果手里只有旧版备份 .db 文件，可以在网页左侧“数据工具”里选择“导入旧版 .db 备份”。
14. 后续升级或迁移到云端时，也以 data 文件夹作为导入来源。
15. 两台电脑不要分别双击运行两套工具；只在一台电脑上双击打开工具，另一台电脑用浏览器打开页面里的“手机/其他电脑访问”地址。
16. 手机和电脑连同一个 Wi-Fi 时，可以扫电脑页面里的“手机/其他电脑访问”二维码。
17. 如果 Windows 防火墙提示，允许在专用网络访问。
18. 自动更新失败时，可在“软件更新”卡片点“手动下载安装包”；浏览器暂时打不开时，等 30 秒再双击桌面图标。
19. 如果密码忘记或重启后无法登录，打开 SupportFiles 文件夹，双击 ResetJeffOrderToolPassword.exe，然后重新设置密码；订单数据不会删除。
20. 如果提示启动超时，把 logs 文件夹里的 server.log 和 updater.log 发给开发人员。

请不要删除 server、runtime、data、SupportFiles 文件夹。
