Jeff订单工具使用说明

1. 双击“打开Jeff订单工具.exe”打开工具。
2. 工具会自动启动后台服务，并打开浏览器页面。
3. 用完后可以直接关闭浏览器；需要关闭后台服务时，打开 SupportFiles 文件夹，双击 CloseJeffOrderTool.exe。
4. 数据保存在当前文件夹的 data 目录。
5. 备份时复制整个 data 文件夹，不要只复制单个 orders.db。
6. 后续升级或迁移到云端时，也以 data 文件夹作为导入来源。
7. 手机和电脑连同一个 Wi-Fi 时，可以扫电脑页面里的“手机访问”二维码。
8. 如果 Windows 防火墙提示，允许在专用网络访问。
9. 如果提示启动超时，把 logs 文件夹里的 server.log 发给开发人员。

请不要删除 server、runtime、data、SupportFiles 文件夹。
