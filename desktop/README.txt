Jeff订单工具使用说明

1. 双击“打开Jeff订单工具.exe”打开工具。
2. 工具会自动启动后台服务，并打开浏览器页面。
3. 第一次打开会提示设置管理员密码；以后电脑和手机访问都先输入这个密码。
4. 登记订单时先输入订单号，再选择公司和工厂，最后保存到订单总表。
5. 用完后可以直接关闭浏览器；需要关闭后台服务时，打开 SupportFiles 文件夹，双击 CloseJeffOrderTool.exe。
6. 数据保存在当前文件夹的 data 目录。
7. 备份时复制整个 data 文件夹，不要只复制单个 orders.db。
8. 后续升级或迁移到云端时，也以 data 文件夹作为导入来源。
9. 两台电脑不要分别双击运行两套工具；只在一台电脑上双击打开工具，另一台电脑用浏览器打开页面里的“手机/其他电脑访问”地址。
10. 手机和电脑连同一个 Wi-Fi 时，可以扫电脑页面里的“手机/其他电脑访问”二维码。
11. 如果 Windows 防火墙提示，允许在专用网络访问。
12. 如果密码忘记或重启后无法登录，打开 SupportFiles 文件夹，双击 ResetJeffOrderToolPassword.exe，然后重新设置密码；订单数据不会删除。
13. 如果提示启动超时，把 logs 文件夹里的 server.log 发给开发人员。

请不要删除 server、runtime、data、SupportFiles 文件夹。
